# M-DEXQR

Utilizando la tecnología **DEXQR** de Muridae para potenciar la manera en que los usuarios interactúan con mi negocio mediante **QRs inteligentes** en la **Muridae platform**.

## Tipos de QR incluidos:

- Menú: para mostrar menús de productos o servicios.
- Encuesta: para recolectar opiniones de los usuarios.
- Productos: para mostrar información de productos.
- Eventos: para agendar o mostrar eventos.
- Usuario: personalizados para información de clientes o miembros.

## Estructura

- flows/ → JSON que define cada QR y su comportamiento dentro de la app.
- assets/ → imágenes y recursos gráficos utilizados en los QR.
